<h1 align="center">Conversor de Monedas - Challenge Alura</h1>
<p>Este proyecto es una aplicacion de consolas desarrollada en java que permite realizar la conversion de monedas</p>

<h2>Funcionalidades</h2>

  - Conversion en tiempo real: Obtiene las tasas de cambio actualizadas directamente desde una API
  - Soporte multi-moneda: Permite convertir entre Dólar (USD), Peso Argentino (ARS), Real Brasileño (BRL) y Boliviano (BOB). 
  - Formateo de moneda: Resultados mostrados con dos decimales para mayor precisión financiera. 


<h2>Tecnologias utilizadas</h2>

  - Java 17
  - Gson 2.13.2
  - HttpClient
  - Intellij IDEA


<h2> Estructura delProyecto </h2>

  - Principal.java : Menu interactivo y logica de interaccion
  - ConsultaApi.java : Comunicacion con la API y manejo de Excepciones
  - Moneda.java : Un Recordde Java para el mapeo de JSON
